Roam API

Secure proxy and telemetry service for the Roam Extension

POST /detect

Proxies detection payload (text + screenshot) to the Gemini API to identify travel destinations.

Headers
x-device-id: string (Required)
Request Body
{
  "text": "string",
  "screenshot": "string (optional)",
  "postId": "string",
  "platform": "instagram | tiktok"
}
Response
{
  "isTravel": true,
  "destinations": [...]
}
POST /flights

Proxies live flight searches to Skyscanner and polls for completion.

Headers
x-device-id: string (Required)
Request Body
{
  "iataCode": "string",
  "homeAirport": "string",
  "currency": "string",
  "maxPolls": "number"
}
Response
{
  "price": 199,
  "currency": "EUR",
  "airline": "Ryanair",
  ...
}
POST /events

Saves engagement events to MongoDB and streams them to Power BI (V5 feature).

Headers
x-device-id: string (Required)
Request Body
{
  "events": [
    {
      "eventType": "dwell",
      "destination": "Paris",
      "duration": 5000,
      ...
    }
  ]
}
Response
{
  "success": true,
  "inserted": 1
}
POST /scores

Upserts recalculated intent scores to MongoDB and streams them to Power BI.

Headers
x-device-id: string (Required)
Request Body
{
  "scores": [
    {
      "destination": "Paris",
      "score": 85,
      ...
    }
  ]
}
Response
{
  "success": true,
  "updated": 1
}
GET /scores

Retrieves a device's saved intent scores for cross-device synchronization.

Headers
x-device-id: string (Required)
Response
{
  "scores": [...],
  "lastUpdated": 1713398400000
}
POST /summary

Requests a natural language intent summary from Gemini for the Intent Tab narrative.

Headers
x-device-id: string (Required)
Request Body
{
  "prompt": "string",
  "maxTokens": "number (optional)",
  "temperature": "number (optional)"
}
