# Roam — powered by Skyscanner
> Passive travel intent detection + behavioural interest scoring for Instagram and TikTok web.

## What's New in V4

V1: Detected travel content from captions.
V2: Added live Skyscanner flight prices + wishlist.
V3: Behavioural engagement tracking + Intent Score algorithm.
V4: Centralised Express backend + MongoDB Atlas. API keys moved server-side — users install and go, zero configuration required.

---

## Core Concept

> You don't know you want to go to Japan. But you've spent 4 minutes watching Japan reels, paused on 7 posts, and turned the sound on twice. Roam knows before you do.

V4 makes Roam a real product. No API keys in the extension. No setup. Install, scroll, discover.

## Architecture

```
[Chrome Extension - user's browser]
  - DOM watching + engagement tracking (local)
  - Intent scoring algorithm (local, scorer.ts)
  - chrome.storage.local for UI state + cache
  - Anonymous device UUID for identity
        ↓ HTTPS
[Express Backend - Railway]
  - Holds ALL API keys (Gemma 4, Skyscanner)
  - Proxies detection requests to Gemma 4
  - Proxies flight requests to Skyscanner
  - Stores engagement events in MongoDB
  - Stores intent scores in MongoDB
  - Returns aggregated scores across sessions
        ↓
[MongoDB Atlas]
  - events collection — raw engagement events
  - scores collection — computed intent scores
  - wishlist collection — saved destinations
```

## How It Works (Full V4 Flow)

```
User scrolling Instagram/TikTok in browser
        ↓
[content.ts] DOM watcher detects travel post
  - Extracts caption, hashtags, location tags
  - Starts engagement timer for this post
        ↓
[content.ts] Engagement tracker runs in parallel
  - Measures dwell time (ms on this post)
  - Detects: rewatch, sound toggle, profile click,
    hashtag click, scroll-back, save, share,
    comment open, caption expand, video pause
  - Emits EngagementEvent to background on scroll-away
        ↓
[background.ts] receives two streams:
  Stream A: DetectionPayload → POST /detect → Gemma 4 → destination + vibes
  Stream B: EngagementEvent → scorer.ts (local) → POST /events → MongoDB
        ↓
[backend] stores events, recomputes scores, returns updated InterestScore[]
        ↓
[popup.ts] three tabs:
  Feed    → live detections with flight prices
  Intent  → destinations ranked by intent score with breakdown + NL summary
  Saved   → starred wishlist
```

## Backend — Express Routes

### POST /detect
Proxies caption text to Gemma 4. Extension never touches the Gemma API key.

**Request:**
```json
{
  "text": "caption + hashtags from DOM",
  "postId": "unique post identifier",
  "platform": "instagram"
}
```

**Response:**
```json
{
  "isTravel": true,
  "destination": "Mua Cave",
  "country": "Vietnam",
  "countryCode": "VN",
  "iataCode": "HAN",
  "vibes": ["Nature", "Adventure", "Off-the-beaten-path"]
}
```

### POST /flights
Proxies IATA code to Skyscanner API.

**Request:**
```json
{
  "iataCode": "HAN",
  "currency": "GBP",
  "homeAirport": "LHR"
}
```

**Response:**
```json
{
  "price": 436,
  "currency": "GBP",
  "airline": "Qatar Airways",
  "duration": "16h 15m",
  "deeplink": "https://skyscanner.net/..."
}
```

### POST /events
Stores raw engagement events in MongoDB.

**Request:**
```json
{
  "deviceId": "uuid-v4",
  "events": [
    {
      "destination": "Mua Cave",
      "countryCode": "VN",
      "eventType": "dwell",
      "duration": 44200,
      "postId": "abc123",
      "platform": "instagram",
      "timestamp": 1714000000000
    }
  ]
}
```

### POST /scores
Upserts computed intent scores for a device.

**Request:**
```json
{
  "deviceId": "uuid-v4",
  "scores": [InterestScore]
}
```

### GET /scores/:deviceId
Returns all intent scores for a device — enables cross-browser, cross-device sync.

**Response:**
```json
{
  "scores": [InterestScore],
  "lastUpdated": 1714000000000
}
```

## Backend — index.ts

```typescript
import express from 'express';
import cors from 'cors';
import { connectMongo } from './db/mongo';
import detectRouter from './routes/detect';
import flightsRouter from './routes/flights';
import eventsRouter from './routes/events';
import scoresRouter from './routes/scores';

const app = express();

app.use(cors({ origin: '*' })); // extension origin
app.use(express.json());

app.use('/detect', detectRouter);
app.use('/flights', flightsRouter);
app.use('/events', eventsRouter);
app.use('/scores', scoresRouter);

app.get('/health', (_, res) => res.json({ status: 'ok' }));

  app.listen(PORT, () => console.log(`Roam backend on :${PORT}`));
});
```

## Extension — V4 Changes

### Device ID generation (background.ts)
```typescript
async function getDeviceId(): Promise<string> {
  const stored = await chrome.storage.local.get('roam:deviceId');
  if (stored['roam:deviceId']) return stored['roam:deviceId'];
  const id = crypto.randomUUID();
  await chrome.storage.local.set({ 'roam:deviceId': id });
  return id;
}
```

### Backend URL config (background.ts)
```typescript
const BACKEND_URL = 'https://roam-backend.railway.app'; // or localhost:3000

async function detectDestination(payload: DetectionPayload): Promise<DetectionResult> {
  const deviceId = await getDeviceId();
  const res = await fetch(`${BACKEND_URL}/detect`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'x-device-id': deviceId
    },
    body: JSON.stringify(payload)
  });
  return res.json();
}
```

## Scoring Algorithm (unchanged from V3, runs locally)

```typescript
// scorer.ts — pure function, no backend dependency

const WEIGHTS = {
  save: 35,
  share: 30,
  profile_click: 30,
  rewatch: 25,
  hashtag_click: 20,
  like: 20,
  scroll_back: 18,
  comment_open: 15,
  sound_on: 12,
  caption_expand: 10,
  video_pause: 8,
};

const DWELL_RATES = {
  tier1: { max: 5000, rate: 0.5 },    // 0-5s
  tier2: { max: 15000, rate: 0.75 },  // 5-15s
  tier3: { max: 30000, rate: 1.0 },   // 15-30s
  tier4: { rate: 1.5 },               // 30s+
};

const MULTI_POST_BONUS = 10;
const DECAY_HALF_LIFE_MS = 7 * 24 * 60 * 60 * 1000;

export function calculateDwellScore(durationMs: number): number {
  const s = durationMs / 1000;
  if (s <= 5) return s * DWELL_RATES.tier1.rate;
  if (s <= 15) return 5 * 0.5 + (s - 5) * DWELL_RATES.tier2.rate;
  if (s <= 30) return 5 * 0.5 + 10 * 0.75 + (s - 15) * DWELL_RATES.tier3.rate;
  return 5 * 0.5 + 10 * 0.75 + 15 * 1.0 + (s - 30) * DWELL_RATES.tier4.rate;
}

export function calculateRawScore(events: EngagementEvent[]): number {
  const now = Date.now();
  const seenPosts = new Set<string>();

  return events.reduce((total, event) => {
    const age = now - event.timestamp;
    const decay = Math.pow(0.5, age / DECAY_HALF_LIFE_MS);

    if (event.eventType === 'dwell' && event.duration) {
      const bonus = !seenPosts.has(event.postId) ? MULTI_POST_BONUS : 0;
      seenPosts.add(event.postId);
      return total + (calculateDwellScore(event.duration) + bonus) * decay;
    }

    return total + (WEIGHTS[event.eventType] ?? 0) * decay;
  }, 0);
}

export function normaliseScores(scores: Map<string, number>): Map<string, number> {
  const max = Math.max(...scores.values());
  if (max === 0) return scores;
  const normalised = new Map<string, number>();
  scores.forEach((raw, dest) => {
    normalised.set(dest, Math.round((raw / max) * 100));
  });
  return normalised;
}
```
